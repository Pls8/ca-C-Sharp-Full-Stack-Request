دليل CSS الشامل

١. ثلاث طرق لتطبيق CSS

CSS يمكن تطبيقه بثلاث طرق مختلفة، لكل منها استخداماته الخاصة ومستويات الأولوية.

النمط المضمن (Inline Style)

يتم تطبيق النمط مباشرة على العنصر باستخدام سمة style. له أعلى أولوية ولكن لا ينصح باستخدامه بكثرة.
<p style="color: red; font-size: 16px;">هذا نص أحمر</p>

النمط الداخلي (Internal Style)

يتم وضع CSS داخل وسم <style> في رأس HTML. ينطبق على المستند بأكمله.
<style> p { color: blue; font-size: 18px; } </style>

النمط الخارجي (External Style)

يتم وضع CSS في ملف منفصل ثم ربطه بـ HTML. هذه أفضل ممارسة للمشاريع الكبيرة.
<link rel="stylesheet" href="styles.css">
/* في ملف styles.css */ p { color: green; font-size: 20px; }

٢. محددات CSS (Selectors)

المحددات تستخدم لتحديد العناصر التي تريد تطبيق الأنماط عليها. هناك عدة أنواع من المحددات:
#idName { }
/* تحديد عنصر بواسطة المعرف (فريد) */
.className { }
/* تحديد عناصر بواسطة الفئة (متعددة) */
p { }
/* تحديد جميع عناصر النوع */
div.className { }
/* عنصر مع فئة محددة */
/* أمثلة متقدمة على المحددات */ div p { } /* جميع عناصر p داخل div */ div > p { } /* عناصر p المباشرة داخل div */ p:hover { } /* عند تمرير الماوس فوق p */ p:first-child { } /* أول عنصر p */

٣. التعليقات في CSS

التعليقات تستخدم لإضافة ملاحظات وشرح للكود دون التأثير على التصميم.
/* هذا تعليق من سطر واحد */ /* هذا تعليق متعدد الأسطر */ p { color: red; /* تعليق بجانب الخاصية */ }

٤. الألوان في CSS

يمكن تحديد الألوان في CSS بعدة طرق مختلفة:
color: red;
/* أسماء الألوان */
color: #ff0000;
/* نظام HEX */
color: rgb(255, 0, 0);
/* نظام RGB */
color: rgba(255, 0, 0, 0.5);
/* RGB مع الشفافية */
نصيحة: استخدم نظام HEX أو RGB للألوان للحصول على تحكم دقيق.

٥. خصائص الخطوط

خصائص الخطوط تتحكم في مظهر النص وحجمه ونوعه.
p { font-family: Arial, sans-serif; /* نوع الخط */ font-size: 16px; /* حجم الخط */ font-weight: bold; /* سماكة الخط */ font-style: italic; /* نمط الخط */ line-height: 1.5; /* ارتفاع السطر */ text-transform: uppercase; /* تحويل النص */ }

٦. استيراد خطوط جوجل

يمكن استيراد خطوط من Google Fonts لاستخدامها في موقعك.
<!-- في رأس HTML --> <link href="https://fonts.googleapis.com/css2?family=Tajawal:wght@400;700&display=swap" rel="stylesheet">
/* في CSS */ body { font-family: 'Tajawal', sans-serif; }
يمكنك استكشاف المزيد من الخطوط على Google Fonts

٧. خصائص النص

خصائص النص تتحكم في تنسيق ومظهر النصوص.
p { text-align: center; /* محاذاة النص */ text-decoration: underline; /* تزيين النص */ text-transform: uppercase; /* تحويل حالة النص */ line-height: 1.6; /* ارتفاع السطر */ letter-spacing: 2px; /* تباعد الحروف */ word-spacing: 5px; /* تباعد الكلمات */ text-indent: 20px; /* مسافة بادئة */ direction: rtl; /* اتجاه النص */ }

٨. وحدات القياس في CSS

هناك نوعان رئيسيان من وحدات القياس: مطلقة ونسبية.

الوحدات المطلقة (ثابتة)

px /* بكسل */ cm /* سنتيمتر */ mm /* مليمتر */ in /* بوصة */ pt /* نقطة */

الوحدات النسبية (أفضل للاستجابة)

% /* نسبة مئوية من العنصر الأب */ em /* نسبة من حجم خط العنصر */ rem /* نسبة من حجم خط العنصر الجذر */ vw /* 1% من عرض viewport */ vh /* 1% من ارتفاع viewport */ vmin /* 1% من أصغر بُعد */ vmax /* 1% من أكبر بُعد */
الأفضل استخدام: الوحدات النسبية مثل rem و vw لتصميم متجاوب.

٩. العرض، الارتفاع والتجاوز

تتحكم هذه الخصائص في أبعاد العناصر وكيفية تعاملها مع المحتوى الزائد.
.container { width: 100%; /* نسبة من العنصر الأب */ height: 50vh; /* 50% من ارتفاع النافذة */ min-width: 300px; /* أقل عرض مسموح */ max-width: 1200px; /* أكبر عرض مسموح */ /* التعامل مع المحتوى الزائد */ overflow: visible; /* إظهار المحتوى الزائد */ overflow: hidden; /* إخفاء المحتوى الزائد */ overflow: scroll; /* إضافة أشرطة التمرير */ overflow: auto; /* أشرطة تمرير عند الحاجة */ }

١٠. خصائص الخلفية

خصائص الخلفية تتحكم في مظهر خلفية العناصر.
div { background-color: #ffffff; background-image: url('image.jpg'); background-size: cover; /* cover, contain, أو قيم محددة */ background-repeat: no-repeat; /* repeat, no-repeat, repeat-x, repeat-y */ background-position: center; /* top, bottom, left, right, center */ background-attachment: fixed; /* scroll, fixed */ /* الاختصار */ background: #fff url('image.jpg') no-repeat center/cover fixed; }

١١. التدرجات اللونية

التدرجات اللونية تسمح بإنشاء خلفيات متعددة الألوان بشكل سلس.
/* تدرج خطي */ background: linear-gradient(to right, red, blue); /* تدرج دائري */ background: radial-gradient(circle, red, yellow, green); /* تدرج بزاوية */ background: linear-gradient(45deg, red, orange, yellow); /* تدرج متكرر */ background: repeating-linear-gradient(45deg, red, red 10px, blue 10px, blue 20px);

١٢. خاصية العرض (Display)

خاصية display تتحكم في كيفية عرض العنصر وتخطيطه.
display: block;
/* يأخذ العرض كامل ويبدأ سطر جديد */
display: inline;
/* يأخذ المساحة اللازمة فقط */
display: inline-block;
/* مثل inline لكن يمكن تحديد أبعاد */
display: none;
/* إخفاء العنصر completamente */
display: flex;
/* تخطيط flexbox */
display: grid;
/* تخطيط grid */

١٣. أدوات المطور (Developer Tools)

أدوات المطور في المتصفح تسمح لك بفحص وتعديل CSS في الوقت الحقيقي.
/* كيفية الوصول للأدوات */ - انقر بزر الماوس الأيمن → "فحص" (Inspect) - أو اضغط F12 - أو Ctrl+Shift+I (Windows/Linux) - أو Cmd+Opt+I (Mac) /* الميزات المتاحة */ - عرض وتحرير CSS مباشرة - تصحيح مشاكل التخطيط - اختبار التصميم المتجاوب - فحص الأنماط المحسوبة - محاكاة الأجهزة المختلفة
هذه الأدوات ضرورية لأي مطور ويب وتوفر وقتًا كبيرًا في التصحيح.

١٤. دالة calc()

دالة calc() تسمح بإجراء حسابات رياضية للقيم في CSS، ويمكنها دمج وحدات قياس مختلفة.
.element { width: calc(100% - 50px); height: calc(50vh + 2rem); margin: calc(2 * 10px); font-size: calc(16px + 0.5vw); padding: calc((100% - 20px) / 3); }
يمكن استخدام العمليات: +, -, *, / مع أي وحدات قياس.

١٥. الحدود والإطار (Border & Outline)

الحدود والإطار يضيفان تزيينًا حول العناصر، لكن لهما استخدامات مختلفة.
.box { /* الحدود */ border: 2px solid red; /* عرض نمط لون */ border-radius: 10px; /* زوايا دائرية */ border-bottom: 1px dotted blue; /* حد سفلي فقط */ border-top-right-radius: 5px; /* زاوية محددة */ /* الإطار */ outline: 2px dashed green; /* خارج الحدود */ outline-offset: 5px; /* مسافة بين الحدود والإطار */ }
الفرق: الإطار خارج الحدود ولا يأخذ مساحة، بينما الحدود جزء من نموذج الصندوق.

١٦. نموذج الصندوق (Box Model)

كل عنصر في HTML يعامل كصندوق يتكون من: المحتوى، الحشو، الحدود، والهوامش.
/* من الداخل إلى الخارج: */ المحتوى → الحشو → الحدود → الهوامش div { width: 200px; /* عرض المحتوى */ padding: 20px; /* مساحة داخل الحدود */ border: 2px solid black; /* الحدود حول الحشو */ margin: 10px; /* مساحة خارج الحدود */ }
العرض الكلي = width + padding + border + margin

١٧. خاصية box-sizing

تتحكم في كيفية حساب أبعاد العنصر - هل تشمل الحشو والحدود أم لا.
/* الافتراضي - العرض والارتفاع للمحتوى فقط */ box-sizing: content-box; /* الأفضل - العرض والارتفاع يشملان الحشو والحدود */ box-sizing: border-box;
النصيحة: استخدم * { box-sizing: border-box; } في بداية CSS لتسهيل التخطيط.

١٨. العناصر الزائفة (Pseudo-elements)

تسمح بتصميم أجزاء محددة من العنصر أو إضافة محتوى زائف.
/* إضافة محتوى قبل العنصر */ p::before { content: "→ "; color: red; font-weight: bold; } /* إضافة محتوى بعد العنصر */ p::after { content: "!"; font-weight: bold; color: blue; } /* تنسيق السطر الأول */ p::first-line { font-weight: bold; color: purple; } /* تنسيق الحرف الأول */ p::first-letter { font-size: 2em; color: red; float: right; margin-left: 5px; }

١٩. التحريك والإطارات الرئيسية

تسمح بإنشاء حركات وتأثيرات متقدمة على العناصر.
/* تعريف التحريك */ @keyframes example { 0% { background-color: red; transform: translateX(0); } 50% { background-color: yellow; transform: translateX(100px); } 100% { background-color: green; transform: translateX(0); } } /* تطبيق التحريك */ div { animation: example 3s ease-in-out infinite; /* أو بشكل مفصل */ animation-name: example; animation-duration: 3s; animation-timing-function: ease-in-out; animation-iteration-count: infinite; animation-direction: alternate; }

٢٠. التحولات (Transitions)

تجعل التغيرات في الخصائص سلسة ومتحركة بدلاً من أن تكون فورية.
.button { background-color: blue; transition: all 0.3s ease-in-out; /* أو بشكل مفصل */ transition-property: all; transition-duration: 0.3s; transition-timing-function: ease-in-out; transition-delay: 0.1s; } .button:hover { background-color: red; transform: scale(1.1); }

٢١. أيقونات Font Awesome

مكتبة رائعة لإضافة أيقونات جميلة لموقعك بسهولة.
<!-- تضمين Font Awesome --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css"> <!-- استخدام الأيقونات --> <i class="fas fa-heart"></i> <!-- نمط صلب --> <i class="far fa-heart"></i> <!-- نمط عادي --> <i class="fab fa-twitter"></i> <!-- نمط العلامات --> <!-- مع CSS --> .icon { color: #ff6b6b; font-size: 24px; margin: 0 10px; }
يمكنك استكشاف جميع الأيقونات على Font Awesome

٢٢. استعلامات الوسائط (Media Queries)

تسمح بتطبيق أنماط مختلفة بناءً على خصائص الجهاز أو المتصفح.
/* للشاشات الصغيرة (هواتف) */ @media screen and (max-width: 768px) { body { background: lightblue; font-size: 14px; } .container { padding: 10px; } } /* للطباعة */ @media print { .no-print { display: none; } body { color: black; background: white; } } /* نقاط التوقف الشائعة */ @media (max-width: 1200px) { /* أجهزة الكمبيوتر المحمولة */ } @media (max-width: 768px) { /* أجهزة لوحية */ } @media (max-width: 480px) { /* هواتف */ } /* التوجيه */ @media (orientation: landscape) { /* أفقي */ } @media (orientation: portrait) { /* عمودي */ }
استعلامات الوسائط أساسية لإنشاء مواقع متجاوبة تعمل على جميع الأجهزة.
