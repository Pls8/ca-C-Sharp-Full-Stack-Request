CSS Cheat Sheet

1. Three Ways to Apply CSS

النمط المضمن

<p style="color: red; font-size: 16px;">This is red text</p>

Internal Style

<style> p { color: blue; font-size: 18px; } </style>

External Style

<link rel="stylesheet" href="styles.css">
p { color: green; font-size: 20px; }

2. CSS Selectors

#idName { } /* Select by ID (unique) */ .className { } /* Select by class (multiple) */ p { } /* Select by tag name */ div.className { } /* Element with specific class */

3. CSS Comments

/* This is a single-line comment */ /* This is a multi-line comment */

4. CSS Colors

color: red; /* Named color */ color: #ff0000; /* HEX color */ color: rgb(255, 0, 0); /* RGB color */ color: rgba(255, 0, 0, 0.5); /* RGB with opacity */ color: hsl(0, 100%, 50%); /* HSL color */

5. CSS Font Properties

font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; /* normal, bold, 100-900 */ font-style: italic; /* normal, italic */ line-height: 1.5;

6. Google Font Import

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
font-family: 'Roboto', sans-serif;

7. CSS Text Properties

text-align: center; /* left, right, center, justify */ text-decoration: underline; /* none, underline, line-through */ text-transform: uppercase; /* lowercase, uppercase, capitalize */ line-height: 1.5; letter-spacing: 2px; text-indent: 20px;

8. CSS Units

Absolute Lengths (Fixed)

px - pixels cm - centimeters mm - millimeters in - inches

Relative Lengths (Better for responsiveness)

% - percentage of parent em - relative to font-size rem - relative to root font-size vw - 1% of viewport width vh - 1% of viewport height
Better to use: Relative units for responsive design

9. Width, Height & Overflow

width: 100%; /* Percentage of parent */ height: 50vh; /* 50% of viewport height */ width: auto; /* Browser calculates */ /* Overflow handles content that doesn't fit */ overflow: visible; /* Default */ overflow: hidden; /* Hide extra content */ overflow: scroll; /* Add scrollbars */ overflow: auto; /* Scrollbars when needed */

10. Background Properties

background-color: #ffffff; background-image: url('image.jpg'); background-size: cover; /* cover, contain, specific size */ background-repeat: no-repeat; /* repeat, no-repeat */ background-position: center; /* top, bottom, left, right, center */ background-attachment: fixed; /* scroll, fixed */ /* Shorthand */ background: #fff url('image.jpg') no-repeat center/cover fixed;

11. Background Gradient

/* Linear Gradient */ background: linear-gradient(to right, red, blue); /* Radial Gradient */ background: radial-gradient(circle, red, yellow, green); /* Multiple color stops */ background: linear-gradient(45deg, red, orange, yellow);

12. CSS Display

display: block; /* Full width, new line (div, p) */ display: inline; /* Only needed width (span, a) */ display: inline-block; /* Inline but can have width/height */ display: none; /* Completely removed */ display: flex; /* Flexbox layout */

13. Browser Inspect Tool

Right-click → "Inspect" or Press F12 - View and edit CSS in real-time - Debug layout issues - Test responsive designs - Check computed styles

14. calc() Method

/* Performs calculations for values */ width: calc(100% - 50px); height: calc(50vh + 2rem); margin: calc(2 * 10px); font-size: calc(16px + 0.5vw);

15. Border and Outline

border: 2px solid red; /* width style color */ border-radius: 10px; /* Rounded corners */ border-bottom: 1px dotted blue; /* Bottom border only */ border-top-right-radius: 5px; /* Specific corner */ outline: 2px dashed green; /* Outside border */ outline-offset: 5px; /* Space between border and outline */

16. Box Model

/* From inside to outside: */ Content → Padding → Border → Margin div { width: 200px; /* Content width */ padding: 20px; /* Space inside border */ border: 2px solid black;/* Border around padding */ margin: 10px; /* Space outside border */ }

17. box-sizing Property

/* Default - width/height = content only */ box-sizing: content-box; /* Better - width/height includes padding & border */ box-sizing: border-box;
Use: * { box-sizing: border-box; } for better layout control

18. Pseudo-elements

/* Add content before element */ ::before { content: "→ "; color: red; } /* Add content after element */ ::after { content: "!"; font-weight: bold; } /* Style first line of text */ ::first-line { font-weight: bold; } /* Style first letter */ ::first-letter { font-size: 2em; color: red; }

19. Animation & Keyframes

/* Define animation */ @keyframes example { 0% { background-color: red; transform: translateX(0); } 50% { background-color: yellow; transform: translateX(100px); } 100% { background-color: green; transform: translateX(0); } } /* Apply animation */ div { animation: example 3s ease-in-out infinite; }

20. Transition

transition: property duration timing-function delay; /* Examples */ transition: all 0.3s ease-in-out; transition: background-color 0.5s linear 0.2s; /* Timing functions */ linear: Constant speed ease-in: Slow start ease-out: Slow end ease-in-out: Slow start and end

21. Font Awesome Icons

<!-- Include Font Awesome --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css"> <!-- Use icons --> <i class="fas fa-heart"></i> Solid style <i class="far fa-heart"></i> Regular style <i class="fab fa-twitter"></i> Brands style

22. Media Queries

/* For screens */ @media screen and (max-width: 768px) { body { background: lightblue; } } /* For printing */ @media print { .no-print { display: none; } } /* Common breakpoints */ @media (max-width: 768px) { /* Tablets */ } @media (max-width: 480px) { /* Phones */ } /* Orientation */ @media (orientation: landscape) { } @media (orientation: portrait) { }
