Event Listeners في JavaScript

✓ 1. حدث النقر (click)

يتم تنشيط هذا الحدث عند النقر على عنصر. وهو الأكثر استخدامًا للأزرار والروابط.

// اختيار العنصر
const button = document.getElementById('myButton');

// إضافة مستمع الحدث
button.addEventListener('click', function() {
  alert('تم النقر على الزر!');
});

✓ 2. حدث الإدخال (input)

يتم تنشيطه عند إدخال نص في حقل إدخال. يعمل مع كل ضغطة زر.

const input = document.getElementById('myInput');

input.addEventListener('input', function(event) {
  console.log('القيمة المدخلة:', event.target.value);
});

✓ 3. حدث التحميل (load)

يتم تنشيطه عند انتهاء تحميل الصفحة أو عنصر معين مثل الصور.

// عند تحميل الصفحة
window.addEventListener('load', function() {
  console.log('تم تحميل الصفحة بالكامل');
});

// عند تحميل صورة
image.addEventListener('load', function() {
  console.log('تم تحميل الصورة');
});

✓ 4. حدث تمرير الفأرة (mouseover)

يتم تنشيطه عند تمرير مؤشر الفأرة فوق عنصر.

const box = document.getElementById('myBox');

box.addEventListener('mouseover', function() {
  this.style.backgroundColor = 'red';
});

box.addEventListener('mouseout', function() {
  this.style.backgroundColor = 'blue';
});

✓ 5. حدث الضغط على مفتاح (keydown)

يتم تنشيطه عند الضغط على أي مفتاح في لوحة المفاتيح.

document.addEventListener('keydown', function(event) {
  if (event.key === 'Enter') {
    console.log('تم الضغط على Enter');
  }
});

✓ 6. حدث إرسال النموذج (submit)

يتم تنشيطه عند إرسال نموذج (form). مهم للتحقق من البيانات قبل الإرسال.

const form = document.getElementById('myForm');

form.addEventListener('submit', function(event) {
  event.preventDefault(); // منع إرسال النموذج
  // التحقق من البيانات
  if (validateForm()) {
    form.submit(); // إرسال النموذج
  }
});
